# _local Scope Overview

## Overview

The `_local` scope contains project-internal decisions specific to how the agentme repository is maintained. These decisions must not be shared with or propagated to other projects. This scope takes the highest precedence and overrides all other scopes.

## Content

### What this scope covers

Project conventions, contributor workflows, repository-specific constraints, and organizational role definitions that apply only within this workspace. The scope also includes research on organizational roles for AI engineering leadership.
