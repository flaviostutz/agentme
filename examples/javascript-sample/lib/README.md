# javascript-sample

A hello world library.

## Quick Start

```bash
pnpm add javascript-sample
```

```typescript
import { hello } from 'javascript-sample';

const greeting = hello('world');
console.log(greeting); // Hello, world!
```

## Installation

```bash
npm install javascript-sample
# or
pnpm add javascript-sample
```

## Examples

See the sibling `examples/` folder for runnable examples that install the packed artifact from `dist/`.

## Development

```bash
make build
make lint
make test
```
